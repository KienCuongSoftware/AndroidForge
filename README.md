AndroidForge is a growing hub of Android Studio projects focused on building real-world mobile apps through clean architecture, modern UI patterns, reusable components, and practical feature development. The repository documents continuous learning, experimentation, and scalable implementation practices for long-term portfolio growth.

## Repository

- GitHub: `https://github.com/KienCuongSoftware/AndroidForge.git`
- Clone: `git clone https://github.com/KienCuongSoftware/AndroidForge.git`

## Repository Structure

- `Lab01_ApplicationComponent`
- `Lab02_UIComponents`
- `Lab03_Layouts`
- `Lab04_Activity`
- `Lab04_Fragment`
- `Lab05_Spinner`
- `Lab05_ListView`
- `Lab05_RecyclerView`
- `Lab05_Onboarding`
- `Lab06_TaskManagerMenu`

## Tech Stack

- Java and Kotlin (depending on project)
- Android SDK
- AndroidX libraries
- Gradle build system

## Getting Started

1. Clone this repository.
2. Open the folder in Android Studio.
3. Select the lab project you want to run.
4. Sync Gradle and run on an emulator or Android device.

## Goal

Build a long-term Android portfolio with practical, incremental projects that improve architecture, UI/UX, and implementation quality over time.
