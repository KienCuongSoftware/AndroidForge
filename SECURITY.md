# Security Policy

## Supported Projects

Security fixes are provided for actively maintained projects in this repository.

## Reporting a Vulnerability

If you discover a security issue, please report it privately to the repository owner instead of creating a public issue.

Repository: `https://github.com/KienCuongSoftware/AndroidForge.git`

Please include:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

## Response Process

After receiving a report, we will:

1. Acknowledge receipt within a reasonable time.
2. Investigate and validate the issue.
3. Prepare and release a fix when confirmed.
4. Credit the reporter if they want public acknowledgment.

## Disclosure

Please avoid public disclosure until a fix is available and users have had time to update.
